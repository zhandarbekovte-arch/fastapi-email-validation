# FastAPI Email Validation API

Бұл жоба студенттік тапсырма ретінде жасалған. Жобаның мақсаты — FastAPI арқылы **API жасау және email деректерін тексеру**.

## Жобаның сипаттамасы

- **FastAPI сервері** — Python арқылы REST API жасауға арналған фреймворк.
- **Pydantic Email Validation** — Email дұрыс жазылғанын автоматты түрде тексереді.
- **GitHub репозиторий** — жобаны интернетке шығарып, басқа адамдармен бөлісу.
- **GitHub Actions** — автоматты тексеру (CI) үшін workflow.

---

## Эндпойнт

### `POST /register`

- Жаңа қолданушыны тіркейді.
- JSON форматында деректерді қабылдайды.

**Request мысалы:**

```json
{
  "name": "Aruzhan",
  "email": "aru@gmail.com"
}
